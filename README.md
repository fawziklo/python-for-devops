# python-for-devops
From Zero Repository for doing python devops

### Build out scaffold:

* Makefile
* requirements.txt
* test_file.py
* python_lib
* Dockerfile
* command-line-tool
* Microservice

1. Create a virtualenv: `virtualenv ~/.venv`
2. edit `~/.bashrc` add `source ~/.venv/bin/activate`
3. clone project then run 'make all'
